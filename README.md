# 🏡 Real Estate Investment Advisor

## 📌 Project Overview
This project predicts:
- ✔ Whether a property is a **Good Investment (Classification)**
- ✔ Expected **Future Price after 5 years (Regression)**

---

## 🚀 Features
- Machine Learning pipelines (Sklearn)
- Feature Engineering for real estate data
- Balanced classification model
- Regression for future price prediction
- Streamlit interactive dashboard
- MLflow experiment tracking

---

## 📊 Dataset Features
- City, State, Locality
- Property Type, BHK
- Size in SqFt
- Price in Lakhs
- Infrastructure (Schools, Hospitals)
- Transport Access
- Amenities

---

## 🧠 ML Models Used
### Classification:
- Random Forest Classifier

### Regression:
- Random Forest Regressor
